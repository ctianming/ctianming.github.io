# Windows 安装 wsl2 最全（大概）踩坑指南

本篇文章全然是本人的经验之谈。我曾因为迫切需要使用 wsl2 却一次又一次地启动失败，哪怕尝试了一种又一种办法，最终却必然卡在重启阶段。

*” 不必担心，我们正在撤销这些操作，请不要关机 “* 几乎成了我的噩梦。

好在，现在它被解决了！

我因为自己本身的问题，搜罗过网络上的许多教程，但毫无疑问的，它们基本都千篇一律，偶尔有对症的，也不够全面。

所以，今日我写下这篇教程，希望帮到和我有一样问题的朋友。

# 先决条件

  • windows 内部版本 19041 及更高版本，你可以输入 win + r 并再次输入 winver 来查看内部版本号
  • 电脑本身支持 Hyper-v 和虚拟化
  • windows 专业版（如果非专业版可能需要手动更新）

# 操作细节

  1. 「Windows 设置 > 更新和安全 > Windows 预览体验计划」处选择加入 “Dev 开发者模式”。
  2. 更新你的 winodws ，直接更新到最新版本。（我曾跳过这步执行下面的操作，但依然被回退）
